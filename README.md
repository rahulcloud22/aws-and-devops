# AWS and DevOps

This repository contains simple examples for AWS and DevOps projects.

| Folder | Description |
| --- | --- |
| [cloudformation](cloudformation) | CloudFormation templates for AWS resources and services. |
| [pulumi](pulumi) | Pulumi examples for infrastructure and serverless apps. |
| [terraform](terraform) | Terraform projects for networking, storage, Kubernetes etc |
| [scripts](scripts) | Helpful scripts for AWS and automation tasks. |

This repo is a simple learning space for cloud infrastructure, automation, and infrastructure as code.

Check out the [notes branch](https://github.com/rahulcloud22/aws-and-devops/tree/notes) for some insights and helpful notes.
